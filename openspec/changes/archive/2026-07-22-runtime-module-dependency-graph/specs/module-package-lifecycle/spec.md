## MODIFIED Requirements

### Requirement: 模块升级必须保留可回滚版本
系统 SHALL 在安装更高且兼容的新版本时保留当前活动版本，并先把新版本完整写入不可变版本目录。系统 MUST 只在整张期望启用模块依赖图存在兼容解时原子提交新的全局激活计划；无解的新版本保留为等待选择的可用版本，当前计划保持不变。

#### Scenario: 升级已安装模块
- **WHEN** 用户安装同一模块 ID 的更高版本，并且新版本可与所有期望启用模块形成兼容组合
- **THEN** 系统原子提交新选择组合，原活动版本保留为可用版本，且上一整套选择计划可用于安全回滚判断

#### Scenario: 协调升级多个等待版本
- **WHEN** 新安装版本使多个先前等待的版本共同形成兼容依赖组合
- **THEN** 系统在一个计划代次中同时切换这些模块，不暴露部分切换状态

#### Scenario: 新版本依赖无解
- **WHEN** 新版本无法与当前已安装版本形成完整兼容组合
- **THEN** 包安装成功但新版本保持等待，当前活动组合不变，管理界面显示依赖诊断

#### Scenario: 拒绝重复或较低版本覆盖
- **WHEN** 用户提交已经存在于该模块版本目录中的版本
- **THEN** 系统拒绝覆盖相同不可变版本；较低但尚未保存的版本只可作为可用版本保存，不得自动替换当前选择

### Requirement: 用户必须能够管理模块生命周期
模块管理界面 SHALL 展示每个运行时模块的选择版本、可用版本、期望启用状态、依赖、依赖者和最近错误，并提供与当前依赖图相符的启用、停用、回滚和卸载操作。系统 MUST 在提交操作前分析直接与间接影响，且不得静默级联修改其他模块。

#### Scenario: 启用等待中的模块
- **WHEN** 用户启用一个依赖尚未满足的模块
- **THEN** 系统保存期望启用状态但不加载入口，并显示等待依赖诊断

#### Scenario: 停用存在依赖者的模块
- **WHEN** 用户尝试停用一个仍被期望启用模块必需依赖的模块
- **THEN** 系统拒绝操作并返回受影响依赖者，现有激活计划保持不变

#### Scenario: 显式安全回滚
- **WHEN** 用户回滚一个存在上一计划版本的模块，且该版本与其他当前选择兼容并不要求隐式改变其他模块
- **THEN** 系统提交新的计划代次，并在重新加载后使用回滚版本

#### Scenario: 回滚会破坏依赖者
- **WHEN** 目标旧版本不满足当前依赖者或需要同时改变其他模块
- **THEN** 系统拒绝单模块回滚、列出受影响模块，并保持当前计划

#### Scenario: 卸载存在依赖者的模块
- **WHEN** 用户确认卸载一个仍被期望启用或等待模块必需依赖的模块
- **THEN** 系统拒绝卸载、返回依赖者列表，并保留该模块的代码与计划状态

#### Scenario: 卸载无依赖者模块
- **WHEN** 用户确认卸载一个没有必需依赖者的运行时模块
- **THEN** 系统从计划和磁盘删除该模块所有版本，并在重新加载后移除其贡献与状态

#### Scenario: 操作失败可见
- **WHEN** 安装、启用、停用、升级、回滚或卸载操作失败或等待依赖
- **THEN** 模块管理界面显示结构化原因，且最后一个完整可用计划保持可用

### Requirement: 安装状态必须可恢复
模块版本目录与模块目录元数据 MUST 先于全局激活计划完整写入。全局计划 MUST 使用防止部分写入的方式保存，并以单调增加的代次标识完整选择组合。进程在包复制、迁移或计划切换期间中断时，下一次启动 SHALL 恢复最后一个完整计划并把额外完整版本视为未选择版本。

#### Scenario: 安装过程中进程中断
- **WHEN** 应用在暂存包内容完成前退出
- **THEN** 下一次启动继续使用原全局计划，并安全清理未完成暂存目录

#### Scenario: 版本落盘后计划写入前中断
- **WHEN** 新版本目录已经完整保存但新激活计划尚未提交时应用退出
- **THEN** 下一次启动继续使用旧计划，并把新版本列为可用但未选择版本

#### Scenario: 计划写入过程中断
- **WHEN** 应用在替换全局激活计划时意外退出
- **THEN** 下一次启动读取旧计划或完整新计划之一，不得读取部分选择组合

#### Scenario: 迁移旧逐模块状态
- **WHEN** 存在 V1 模块 `state.json` 但没有全局激活计划
- **THEN** 系统优先沿用旧活动版本建立初始计划，吸收前端提供的旧禁用模块，并保证重复迁移幂等
