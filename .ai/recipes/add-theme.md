# Add a theme preset

1. Read `src/themes/AGENTS.md`.
2. Add the preset ID to `ThemePresetId`.
3. Register its human-readable metadata in `theme-registry.ts`.
4. Add complete light and dark semantic token blocks to `globals.css`.
5. Do not edit UI primitive classes.
6. Verify common controls, borders, muted text, destructive actions, and keyboard focus in both modes.
